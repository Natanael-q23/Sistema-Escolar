# Sistema-Escolar
Projeto de um sistema escolar produzido em python.
O sistema deverá possuir as funções básicas: Cadastro, Consulta, Atualização e Exclusão dos dados de alunos no Sistema. 
Além das funcionalidades básicas, o software também realzará uma análise de desempenho de cada aluno individualmente.
Com todos os dados e informações dos alunos obtidos, o sistema fará uma análise e mostrará a taxa de aprovação do aluno durante o semestre, ajudando os professores a diferenciar os alunos que estão bem-sucedidos no semestre daqueles que não estão.
